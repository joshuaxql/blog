# zellij介绍
zellij是由rust编写的终端分屏工具，解决了我电脑的windows terminal自带的分屏速度慢，不能快速回复等问题

# 安装

```shell
scoop install main/zellij
```

# 配置

运行
```shell
zellij setup --check
```
获取zellij配置的路径，我的配置文件在[CONFIG DIR]: "C:\Users\用户名\AppData\Roaming\Zellij\config"文件夹下的config.kdl

我们可以先设置默认终端为你喜欢的终端，比如pwsh,zsh,fish等等
```kdl
default_shell "pwsh"
```

然后在[官网配置文档](https://zellij.dev/documentation/theme-list.html)选一个自己喜欢的主题
```kdl
theme "one-half-dark"
```

这样基本的配置就做好了！

# 使用
基本的快捷键都写在底下，很快就能上手。
此外有一个特别的功能，alt+f可以打开一个悬浮terminal，即开即用，非常方便。
